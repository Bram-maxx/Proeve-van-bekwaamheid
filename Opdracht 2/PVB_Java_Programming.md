## Vragen

1. Noem 5 datatypes die je voor variabelen kan gebruiken.

   **Antwoord:** Float, String, Boelian, char, intergar

   <br><br>

2. Noem 2 andere soorten lijsten/collecties in Processing (naast de standaard array met `[]`).

   **Antwoord:** Float en string

   <br><br>

3. Schrijf een standaard *For* loop.

   **Antwoord:** for(int i=1; i <=10; i++)
   {
      println(i);
   }


   <br><br>

4. Schrijf 3 wiskundige operators Bv. % (de modulus zelf mag niet)

   **Antwoord:** +,  -,   *,   /

   <br><br>

5. Schrijf 3 voorwaarde operators plus hoe ze heten Bv. > (de groter dan zelf mag niet)

   **Antwoord:** < kleiner dan
                 == gelijk aan
                 >= groter dan of gelijk aan

   <br><br>

6. Wat doet ++?

   **Antwoord:** verhoogt een variabele met 1 er elke keer boven op

   <br><br>

7. Wat is het antwoord op 10%3?

   **Antwoord:**  1 
                  om dat 3 past 3 keer in de 10 en blijft "1" achter

   <br><br>

8. Zet achter elke regel True of False.

   - 5 < 3 : false
   - 8 >= 8 :  true
   - 6 > 12 || 6 == 6 : false | true
   - (8*5) > 25 : true
   - AgeRick < 40 : true
